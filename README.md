# Backend for Walrus Manager: An Android Schedule Management Application

## Running the project
1. Make sure the datasource in application.yml are correct
2. run WalrusApplication.java
3. Open localhost:8080 in Browser

## See API document
1. Make sure application is running
2. Open http://localhost:8080/swagger-ui/index.html
